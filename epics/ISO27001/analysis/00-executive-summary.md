## Critical Success Factors

### Must-Haves for Success

1. **Executive Commitment**
   - Active CEO/executive sponsorship
   - Budget approval and resource allocation
   - Policy approval and communication
   - **Risk if missing:** Initiative stalls, audits fail

2. **Dedicated Security Leadership**
   - Named person with accountability and authority
   - Virtual CISO minimum (0.3-0.5 FTE)
   - **Risk if missing:** No coordination, fragmentation

3. **Cloud Provider Leverage**
   - Maximize inherited controls from certified providers
   - Obtain attestations and document shared responsibility
   - **Risk if missing:** 30-40% more effort and cost

4. **Pragmatic Scoping**
   - Scope ISMS appropriately (exclude out-of-scope systems)
   - Focus on client-facing systems and data processing
   - **Risk if missing:** Unnecessary complexity and cost

5. **Template & Framework Use**
   - Don't reinvent the wheel - use proven templates
   - NIST, CIS, ISO frameworks
   - **Risk if missing:** 3-6 months additional documentation time

6. **Realistic Timeline**
   - 12-18 months is realistic for low-maturity organization
   - Avoid crash programs (6 months = high failure risk)
   - **Risk if missing:** Rushed implementation, audit failures, wasted costs
