## DESIGN NOTES (For Visual Version)

When creating the visual/designed version of this Strategy-on-a-Page:

**Layout Principles:**
- Fits on ONE PAGE (legal or 11x17 acceptable, letter preferred)
- Visual hierarchy: Mission/Vision top, themes middle, roadmap bottom
- Color-coding: Each theme gets a color, used consistently
- Icons: Simple icons for each theme
- White space: Don't cram—negative space aids comprehension

**Visual Elements:**
- Timeline/roadmap with phases clearly demarcated
- Metrics dashboard as simple bar chart or thermometer
- Theme "boxes" with consistent structure
- Investment allocation as pie chart or stacked bar

**Typography:**
- Title: Large, bold, memorable
- Theme names: Clear hierarchy, easy to scan
- Metrics: Tabular, easy to read quickly
- Body text: Minimal, bullet points preferred

**Delivery Formats:**
- PDF (for printing and sharing)
- PowerPoint/Keynote (for presentations)
- Poster (for office walls)
- Digital dashboard (interactive version)
